The corporate web presence for HAVEN Intelligence Consulting. Built with Astro 6 for static generation, Tailwind CSS v4 with CSS-first configuration, and deployed to Cloudflare Pages.

## Technical highlights

- Astro 6 with content collections and glob loader
- Tailwind CSS v4 using @theme design tokens
- Full Danish/English i18n with locale-prefixed routing
- Cloudflare Pages static deployment with custom headers
- Structured data (JSON-LD) for SEO
- Scroll-driven animations using the CSS View Transitions API
